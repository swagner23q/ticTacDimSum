Features

Features are flat content that contain an image, a heading, and some description text. They are typically used as a heading for a service page or as a full-width section on another page. Here are some examples of how features can be organized.

Sample Feature
.gfx-left

This feature shows the graphic to the left and the title on the right. This is one of four ways a feature can be displayed. The position of the graphic and the title are determined by their order in the DOM as well as a class applied to the element.

Sample Feature
.gfx-right

Features can always have icons in their title by adding an <i> element inside the .title heading. Also, adding an .odd or .even class to a feature gives it an accent background which is useful for displaying multiple features in a row.

Sample Feature .gfx-top

This feature shows the graphic above and the description below. This is one of four ways a feature can be displayed. With .gfx-top and .gfx-bottom the graphic is styled to be 50% the width of its parent and have auto margins left and right.

Sample Feature .gfx-bottom

This feature shows the graphic below and the description above. The children of feature elements should always be wrapped in a .wrapper element for spacing and alignment.
