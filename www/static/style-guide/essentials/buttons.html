Button Styles

There are seven styles of buttons used on the website. Each style has a default, hover, and active state.

Default
Primary
Success
Info
Warning
Danger
Link

To achieve the button styles above, use the following classes:

  • .btn.default
  • .btn.primary
  • .btn.success
  • .btn.info
  • .btn.warning
  • .btn.danger
  • .btn.link
