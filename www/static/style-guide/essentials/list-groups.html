List Groups

This page depicts the way list groups can be displayed on this website.

Basic List Groups

The most basic list group is an unordered list with list items:

  • First item
  • Second item
  • Third item

To create a basic list group, use an <ul> element with class .list-group, and <li> elements with class .list-group-item.

List Group with Badges

You can also add badges to a list group. The badges will automatically be positioned on the right:

  • 12Boats
  • 2Trains
  • 10Planes

To create a list group with a badge, add a <span> element with class .badge, inside the .list-group-item element.

List Group with Linked Items

The items in a list group can also be hyperlinks. This will add a grey background color on hover:

12Boats2Trains10Planes

To create a list group with linked items, use a <div> element with a class of .list-group instead of a <ul> element. The use <a> elements with classes of .list-group-item instead of <li>.
