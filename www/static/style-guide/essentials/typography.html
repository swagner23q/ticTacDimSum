Typography

This page depicts all typographical elements for use in this website.

<h1> - <h6>

HTML headings (<h1> to <h6>) are styled in the following way:

h1 Heading 1

h2 Heading 2

h3 Heading 3

h4 Heading 4

h5 Heading 5
h6 Heading 6

<small>

The HTML <small> element is used to create a lighter, secondary text in any heading:

h1 Heading 1 secondary text

h2 Heading 2 secondary text

h3 Heading 3 secondary text

h4 Heading 4 secondary text

h5 Heading 5 secondary text
h6 Heading 6 secondary text

<p>

Paragraphs (<p>) are styled in the following way:

There is perhaps no better a demonstration of the folly of human conceits than this distant image of our tiny world.

<ul>, <ol>, <li>

Unordered (<ul>) and ordered (<ol>) lists are styled in the following way:

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  1. Justin Beans
  2. Alexander Hamilton
  3. Jerry Fettes
  4. Hubert Humphery

<mark>

The HTML <mark> element will style text in the following way:

Use the mark element to highlight text.

<abbr>

The HTML <abbr> element for abbreviation will style text in the following way:

The WWF sued the WWF for trademark infringement, forcing the WWF to be renamed to the WWE.

<blockquote>

The HTML <blockquote> element will style text in the following way:

<p> The Saturn V rocket’s first stage carries 203,400 gallons (770,000 liters) of kerosene fuel and 318,000 gallons (1.2 million liters) of liquid oxygen needed for combustion. At liftoff, the stage’s five F-1 rocket engines ignite and produce 7.5 million pounds of thrust.

<footer> - See more at: www.space.com

To show the quote on the right, use the .blockquote-reverse class:

<p> The Saturn V rocket’s first stage carries 203,400 gallons (770,000 liters) of kerosene fuel and 318,000 gallons (1.2 million liters) of liquid oxygen needed for combustion. At liftoff, the stage’s five F-1 rocket engines ignite and produce 7.5 million pounds of thrust.

<footer> See more at: www.space.com

<dl>

The HTML <dl> element for description list will style text in the following way:

<dt> Computer
<dd> A device for digital activities
<dt> Phone
<dd> A device for speaking to people who are not nearby

<code>

The HTML <code> element will style text in the following way:

The following HTML elements: span, section, and div defines a section in a document.

<kbd>

The HTML <kbd> element will style text in the following way:

Use CMD + p to open the Print dialog box.

<pre>

The HTML <pre> element will style text in the following way:

Text in a pre element
is displayed in a fixed-width
font, and it preserves
both      spaces and
line breaks.
