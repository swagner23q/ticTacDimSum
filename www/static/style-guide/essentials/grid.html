The Grid

The foundation of any well formed site is a grid system. The implementation here is using Jeet, a grid system for human. Jeet is available anywhere in stylus and here are some examples of the things you can do with Jeet.

Flexibility

Jeet is built on CSS preprocessors. As such, you're no longer limited to classes like .col-6. Instead, you can pass any fraction, decimal, or combination of the two, to create a completely custom grid.

Helper Classes

Jeet has been extended here to include helper classes that can be added to DOM elements. Elements can be given a fixed column width and cycle by using the following classes: .col-one-half, .col-one-third, .col-one-fourth, .col-one-fifth, etc. onto 12 columns.

Stylus Mixins

Be sure to use Jeet's mixins in elements when you can. col(ratios: 1, offset: 0, cycle: 0, uncycle: 0, gutter: jeet-gutter). Again, this mixin is available anywhere you can write stylus.

Tips

It's a good idea to wrap your columns in an element that is self cleared. This is accomplished by adding a DOM element with the class of .clearfix around your columns or extending the clearfix() mixin directly on your element containing the columns.
